3種類方法がある
- volumes ... 保存場所を別に用意して、コンテナ間でそのファイルを共有する（推奨されている方法）
- bind(名前付き)
- bind(匿名)

## 1ファイルだけコンテナ内にバインドしたい時
以下のように設定する
```
volumes:
  - ./docker/nginx/nginx.conf:/etc/nginx/nginx.conf
```
